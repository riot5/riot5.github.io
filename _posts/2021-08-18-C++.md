##  " c++  1. 기본형태 "

```c++
#include<iostream> // 헤더파일 <입출력 표준기능>  : 헤더 파일을 포함시키틑 기능
  
using namspace std; //namespace 안에 존재 하는 std 사용
 
int main()                     //실행과정 ( 컴파일(고급언어 = C or C++ 등) -> 빌드(저급언어 = 기계어) -> 실행)
{
  std :: cout << "출력할 내용"  
  return 0;
}
```
